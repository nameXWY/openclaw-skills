# GFW（中国防火墙）完整规则

> 最后验证时间：2026-03-06

## 是什么
架设在国内外网络交界处的**过滤/封锁系统**，不是单一设备，是一套系统。

---

## 5种封锁手段

### 1. DNS污染（最常见）
- **原理**：UDP 53端口查询DNS时，返回假的IP地址
- **影响**：访问 google.com → 解析到错误IP → 打不开
- **证据**：2025年研究发现GFW对UDP 53端口进行DNS查询过滤

### 2. IP封锁（路由黑洞）
- **原理**：把目标IP加入黑名单，直接丢弃数据包
- **影响**：某些IP完全无法访问
- **例子**：Google部分IP被封

### 3. 关键字过滤（HTTP明文）
- **原理**：检测HTTP请求/响应中的敏感关键词
- **手段**：发送RST包切断连接
- **影响**：HTTP明文网页可能被干扰

### 4. 端口封锁
- **原理**：封锁特定端口
- **常见被封端口**：
  - 25（SMTP）- 发邮件端口，国内云服务商默认封锁
  - 993（IMAP）- Gmail等邮箱的收信端口
  - 465（SMTPS）- 加密SMTP

### 5. SSL/HTTPS阻断
- **原理**：检测TLS握手中的SNI（Server Name Indication）
- **手段**：发送伪造的RST包切断连接
- **证据**：2025年8月20日曾出现对443端口无差别封锁
- **影响**：Gmail、Google服务长期被干扰

---

## 对常用服务的影响

| 服务 | 状态 | 原因 |
|------|------|------|
| Google搜索 | ❌ | IP封锁+DNS污染 |
| Gmail | ❌ | 993端口被封+SSL阻断 |
| YouTube | ❌ | IP封锁 |
| GitHub | ⚠️ | 部分可用，有时不稳定 |
| OpenAI | ❌ | IP封锁+域名污染 |
| Wikipedia | ❌ | 关键字过滤 |

---

## 对邮件的影响

| 端口 | 协议 | 国内状态 |
|------|------|----------|
| 25 | SMTP | ❌ 被封锁（发件） |
| 465 | SMTPS | ⚠️ 不稳定 |
| 587 | SMTP | ⚠️ 不稳定 |
| 993 | IMAP | ❌ 被封锁（收件） |
| 995 | POP3 | ⚠️ 勉强可用 |

**Gmail**国内无法直接使用（993被封）

---

## 总结

- GFW是**整个网络出口**的过滤系统
- 封锁手段：**DNS污染 + IP封锁 + 关键字过滤 + 端口封锁 + SSL阻断**
- Gmail/Google等**基本全封**
- 国内发邮件到国外**困难**
